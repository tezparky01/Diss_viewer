Enable Viewpoints on Quality Page

Since viewpoints are enabled on the main 3D page but not integrated with quality inspection, we can enhance the quality workflow.

// Add to Quality Panel const viewpoints = components.get(OBC.Viewpoints); // Save inspection viewpoint const saveInspectionView = () => { const viewpoint = viewpoints.create(world); return viewpoint; // Store with inspection data }; // Load saved viewpoint for inspection review const loadInspectionView = (viewpointId) => { viewpoints.load(world, viewpointId); };
